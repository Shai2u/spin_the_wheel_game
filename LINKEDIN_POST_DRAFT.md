## Option 1 (Short)

Built a fun side project for my child: **Spin the Yarin!** 🎡  
A kid-friendly Wheel of Fortune game with bright visuals, drag-and-drop tasks, and bilingual voice output (Hebrew + English).

Tech:
- Django backend
- React frontend (browser-loaded)
- SQLite + local fallback persistence

Features I’m proud of:
- Drag tasks between bank and wheel
- Spin by button or drag gesture
- Needle friction/tension effect
- Text-to-speech winner announcement
- Morning/Dawn/Sunset/Night themes
- Tablet/iPad support over local network

Would love feedback on UX/game polish ideas 🙌

## Option 2 (Story Style)

I wanted a simple game my child could enjoy on tablet, so I built one: **Spin the Yarin!** 🎡✨

The goal was to make chores/activities playful instead of “another to-do list”.

What I implemented:
- A task bank (add/edit/delete)
- Drag-and-drop tasks into wheel slices
- Spin mechanics with randomization + suspenseful needle friction
- Winner highlighting + voice readout (Hebrew/English)
- Day themes (morning, dawn, sunset, night)
- Persistent state with Django API + local fallback

It now runs locally and can be opened from iPad/phone on the same Wi-Fi.

Next ideas:
- PWA install support
- Confetti/sound toggles
- Export/import task packs

If anyone built similar family-friendly products, I’d love to hear what worked for you.
